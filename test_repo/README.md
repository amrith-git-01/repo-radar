# Test Repository

This is a sample Python application used for testing the DevRamp analysis system.

## Overview

A simple application that demonstrates:
- Configuration management
- Utility functions
- Unit testing
- CI/CD workflows

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py
```

## Testing

```bash
pytest tests/
```
